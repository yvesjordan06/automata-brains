ATTENTION!!!
Cette application s'adresse a des personnes ayant des connaissances sur les automates et l'ensemble des 
opérations qui s'appliquent sur ceux ci.

                    ***OUTILS A AVOIR AU PREABLE***

--> Python3.6 à Python3.7
--> Pyqt5
--> graphiz


                     ***COMMENT EXTRAIRE***

1- Faire un clique gauche sur le fichier .zip
2- Cliquer sur "decompresser" ou "extraire ici" vous obtenez un dossier

                    ***COMMENT LANCER L'APPLICATION***

3- Entrer dans le dossier obtenu apres avoir decompresse.Ouvrir un invite de commande la dossier 'view'
4- Lancez l'application en tapant << python main.py>>

                    ***MODE D'UTILISATION***
--> Creer l'automate 
-Creer votre automate en cliquant sur le boutton <<Nouveau>>

--> Entrer le nom de l'automate
-Entrer le nom de votre automate

--> Alphabet
-Entrer un a un les symboles de votre alphabet, apres avoir entre un symbole cliquez sur <<inserer>>

--> Etats
-Entrer un a un vos etats, apres avoir enter un etat clique sur <<initial>> si votre etat est un etat initial
sur <<final>> si c'est un etat final puis sur <<inserer>>. Si votre etat n'est ni initial ni final, entrez
votre etat et cliquez sur <<inserer>>

--> Transitions
NB: une transition c'est depart-symbol-arrive
- Selectionnez un etat de depart
- Selectionnez un symbol ou cliquez sur <<epsilon>> selon le type de votre automate
- Selectionnez un etat d'arrive
- Puis cliquer sur <<Ajouter>>
